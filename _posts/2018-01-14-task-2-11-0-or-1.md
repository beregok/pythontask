Вводиться число 0 або 1, необхідно вивести 1 або 0 відповідно.

## Формат введення

Число 0 или 1.

## Формат виведення

Число 0 или 1.

## Приклади

```bash
Тест 1
0
Виведення програми:
1
```

## Розв'язання

```python
n = int(input())
print((1 - n) // 1)
```
