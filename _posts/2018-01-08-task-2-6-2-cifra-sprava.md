Дано натуральне число. Знайдіть цифру, що стоїть в розряді десятків в його десяткового запису (другу праворуч цифру).

## Формат введення

Вводиться єдине число.

## Формат виведення

Виведіть відповідь до задачі.

## Приклади

```bash
Тест 1
Вхідні дані:
73
Виведення програми:
7
```

## Розв'язання

```python
n = int(input())
print((n // 10) % 10)
```
