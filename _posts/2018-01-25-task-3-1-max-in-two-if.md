Напишіть програму, яка зчитує два цілих числа A і B і виводить найбільше значення з них. Числа - цілі від 1 до 1000.

## Формат введення

Вводяться два числа.

## Формат виведення

Виведіть відповідь до задачі.

## Приклади

```bash
Тест 1
Вхідні дані:
8
5
Виведення програми:
8

Тест 2
Вхідні дані:
5
8
Виведення програми:
8

Тест 3
Вхідні дані:
5
5
Виведення програми:
5
```

## Розв'язання

```python
a = int(input())
b = int(input())
if a < b:
    print(b)
if a > b:
    print(a)
if a == b:
    print(a)
```
